# Cita desbloqueable · Spider-Date

Aplicación web móvil tipo TikTok: planificación de cita + examen de Contabilidad Financiera II.

## Uso

Abre `index.html` en el navegador. Para subirla gratis puedes usar Netlify, Vercel o GitHub Pages.

## Incluye

- Diseño responsive móvil.
- Flujo de cita personalizado.
- Opciones personalizadas en comida, postre, bebida y flores.
- Selección múltiple de flores y extras.
- Examen tipo test de 12 preguntas.
- Reinicio del examen si se falla una pregunta.
- Resumen final y ticket de reserva.
- PWA básica con service worker.
