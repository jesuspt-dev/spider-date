# 🕷️ Spider-Date - Cita Desbloqueable

Aplicación web interactiva: planificación de cita personalizada + examen de Contabilidad Financiera II. El 29 de Julio tienes una cita conmigo.

## ✨ Características

- 🎨 **Diseño responsive** tipo TikTok con animaciones suaves
- 🗓️ **Flujo personalizado** de planificación de cita en 9 pasos
- 💬 **Opciones inteligentes**: selección única, múltiple y entrada de texto
- 📝 **Examen de 12 preguntas** de Contabilidad con dificultad creciente
- 📧 **Envío de respuestas** por email con EmailJS
- 💾 **Persistencia local** - guarda tu progreso en localStorage
- 🌐 **PWA** con service worker para funcionamiento offline
- ♿ **Accesible** - WCAG compliant, compatible con lectores de pantalla
- 🌙 **Modo oscuro** automático según preferencias del sistema
- 🎯 **Validación** de campos requeridos

## 🛠️ Configuración EmailJS (Opcional)

Para que funcione el envío de emails:

1. Ve a [emailjs.com](https://www.emailjs.com)
2. Registrate y conecta tu Gmail
3. Crea un template con nombre `template_spider`
4. Las credenciales ya están en `app.js`

- **Envíos**: EmailJS API

## 🎮 Cómo Jugar

1. **Intro**: Lee la misión
2. **Planificación (9 pasos)**: Personaliza cada aspecto de la cita
3. **Bloqueo**: Se guarda tu selección
4. **Examen (12 preguntas)**: Aprueba Contabilidad
5. **Éxito**: ¡Cita desbloqueada! Se envía por email
6. **Ticket**: Resumen final con todos tus detalles
